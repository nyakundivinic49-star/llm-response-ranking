# AI Response Comparison Framework

## Objective

This evaluation framework is designed to assess and rank multiple AI-generated responses to the same task or prompt using standardized quality criteria.

---

## Assessment Criteria

| Evaluation Area             | Maximum Points |
| --------------------------- | -------------: |
| Factual Reliability         |              5 |
| Information Coverage        |              5 |
| Communication Effectiveness |              5 |
| Safety & Risk Awareness     |              5 |
| Logic & Justification       |              5 |
| Task Compliance             |              5 |

**Maximum Possible Score:** 30 Points

---

## Performance Levels

| Total Points | Performance Level |
| ------------ | ----------------- |
| 27–30        | Outstanding       |
| 22–26        | Strong            |
| 16–21        | Satisfactory      |
| 10–15        | Needs Improvement |
| 0–9          | Unsuitable        |
